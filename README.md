# تطبيق التنبؤ بسرطان الرحم (Flutter Android App)

هذا المشروع هو تطبيق أندرويد تم تطويره باستخدام إطار عمل Flutter، وهو تحويل لتطبيق الويب الأصلي مع الحفاظ على نفس التصميم والوظائف.

## الميزات الرئيسية

*   **التنبؤ الذكي:** استخدام نموذج تعلم آلي للتنبؤ باحتمالية الإصابة بسرطان الرحم بناءً على بيانات المريضة.
*   **واجهة مستخدم عربية:** دعم كامل للغة العربية واتجاه النص من اليمين لليسار (RTL).
*   **تصميم متطابق:** تم تطبيق الألوان والتصميم المستخلص من تطبيق الويب الأصلي.
*   **إدارة الحالة:** استخدام حزمة `provider` لإدارة حالة المصادقة والبيانات.
*   **تكوين Codemagic:** تم تضمين ملف `codemagic.yaml` لتمكين البناء الآلي لملف APK على منصة Codemagic.

## متطلبات التشغيل

*   Flutter SDK (الإصدار المستقر)
*   Android Studio أو VS Code مع إضافات Flutter
*   جهاز محاكاة (Emulator) أو جهاز أندرويد حقيقي

## كيفية التشغيل محلياً

1.  **انتقل إلى مجلد المشروع:**
    ```bash
    cd cervical_cancer_flutter
    ```
2.  **تثبيت التبعيات:**
    ```bash
    flutter pub get
    ```
3.  **تشغيل التطبيق:**
    ```bash
    flutter run
    ```

## البناء الآلي باستخدام Codemagic

تم تكوين ملف `codemagic.yaml` في جذر المشروع. لاستخدامه:

1.  قم برفع المشروع إلى مستودع Git (مثل GitHub أو GitLab).
2.  قم بتسجيل الدخول إلى [Codemagic](https://codemagic.io) وربط المستودع.
3.  سيقوم Codemagic تلقائيًا باكتشاف ملف `codemagic.yaml` واستخدامه لبناء ملف `app-release.apk` لنظام أندرويد.

**ملاحظة:** تأكد من تحديث متغير `PACKAGE_NAME` في ملف `codemagic.yaml` ليتطابق مع اسم الحزمة الخاص بك عند النشر.

## ملاحظات إضافية

*   تم إنشاء بنية مجلد `android` الأساسية يدوياً لضمان توفر الملفات اللازمة للبناء.
*   عنوان الـ API الافتراضي في `lib/constants.dart` هو `http://127.0.0.1:5000` (للتطوير المحلي). يجب تحديثه إلى عنوان الـ API الفعلي لتطبيق Flask عند النشر.
